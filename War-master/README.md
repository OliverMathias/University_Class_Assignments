# War

Oliver Mathias
Instructions to run War:

```
Step 1: Compile
javac *.javac

Step 2: Run
java Simulation X <--- X represents the number of games you'd like to simulate.
ex: java Simulation 50
```

```
Description:
This collection of java scripts, controlled by the Simulation.java file, work together to simulates
n number of games of the card game war. Using two players, a standard deck, and the basic rules for
the card game war, it outputs avgerages, minmums, maximums, and other stats about the games.
```
